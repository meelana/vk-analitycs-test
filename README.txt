Вопрос 1

У тебя есть Набор данных по рекламным кампаниям рекламодателя.
Команда попросила подготовить дашборд по эффективности размещений.

В наборе данных есть следующие поля:

• Period - период открутки рекламной кампании;
• Campaign_id - номер рекламной кампании;
• Amount - затраты на кампанию;
• Shows - количество показов;
• Clicks - количество кликов;
• Conversions - количество конверсий.

Необходимо их проанализировать и подготовить визуализацию в BI-системе.
Команде важно посмотреть и на дополнительные метрики – CTR, CРА.

Для выполнения задания можно использовать BI-систему на выбор - Superset или Power BI.
